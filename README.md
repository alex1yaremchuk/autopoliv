# Autopoliv 360 Demo Map

Статический демо-сайт с полноэкранной картой учеников на Яндекс.Картах.

## Что внутри

- `index.html` - только контейнер карты.
- `app.js` - загрузка Яндекс.Карт, чтение CSV и создание отметок.
- `config.js` - ключ API Яндекса и ссылка на CSV.
- `data/students-demo.csv` - демо-данные на 20 учеников.
- `data/google-sheets-import.csv` - CSV для импорта в Google Sheets.
- `assets/photos/` - 5 локальных SVG-фото для повторного использования.

## Как подключить

1. Укажите ключ в `config.js` в поле `yandexMaps.apiKey`.
2. Если нужен Google Sheets, импортируйте `data/google-sheets-import.csv` в таблицу.
3. Опубликуйте лист в формате CSV.
4. Вставьте опубликованный CSV URL в `config.js` в поле `data.googleSheetsCsvUrl`.
5. Если URL таблицы не задан, сайт использует `data/students-demo.csv`.

Служебная первая строка вида `Изменено:,3/19/2026` поддерживается. Сайт ищет строку заголовков автоматически.
Данные кэшируются в `localStorage`. Если CDN вернет более старую версию CSV, сайт оставит более свежие данные по дате из строки `Изменено:`.

## Колонки CSV

`id,name,city,coords,photo,phone,email,course,stage,description,portfolio_url`

Поле `coords` хранится одной строкой в формате:

`59.940162, 30.227793`

В поле `photo` можно хранить:

- локальный путь, например `./assets/photos/student-portrait-01.svg`
- или полный URL изображения

Телефон в демо записан без ведущего `+`, чтобы Google Sheets не пытался трактовать значение как формулу при вставке.
